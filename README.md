🚀 Hybrid Deep Learning Model (LSTM + CNN)
📌 Overview

This project implements a hybrid deep learning model that combines:

🧠 Long Short-Term Memory (LSTM)

🖼 Convolutional Neural Network (CNN)

The purpose of this hybrid architecture is to leverage the strengths of both models:

LSTM for sequential and temporal pattern learning.

CNN for spatial feature extraction.

This combination improves overall prediction accuracy and model performance.

🏗 Model Architecture

The model consists of:

CNN Layers

Feature extraction

Convolution + Activation

Pooling layers

LSTM Layers

Sequence learning

Long-term dependency handling

Fully Connected Layers

Final classification / prediction

Data preprocessing

📈 Model Advantages

✔ Combines spatial + temporal learning
✔ Higher accuracy compared to single models
✔ Better generalization
✔ Suitable for time-series, signal processing, and image-sequence tasks
